## Installation

Use [Git](https://git-scm.com) to clone this Repository into your server folder.

```git
https://github.com/BolajiAyodeji/hugo101.git
```
```git
cd hugo101
```
```
hugo server -D
```
View site on 

```shell
localhost:1313
```
